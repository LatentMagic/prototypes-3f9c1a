# Brand assets

The shippable output. Regenerated by [../scripts/](../scripts/); the spec is [../circlists-brand.md](../circlists-brand.md).

## Vector (SVG)

- `circlists-mark.svg` — the mark; also the favicon / app-icon source.
- `circlists-wordmark.svg` · `circlists-wordmark-reversed.svg` — wordmark, ink (light) / cream (dark).
- `circlists-lockup.svg` · `circlists-lockup-reversed.svg` — mark + wordmark, ink / cream.

## Raster (from the mark)

- `favicon.ico` — 16·32·48, classic browser tab.
- `favicon-16.png` · `favicon-32.png` — modern `<link rel=icon>`.
- `apple-touch-icon.png` — 180, iOS home screen.
- `icon-192.png` · `icon-512.png` — PWA / Android; the 512 also serves avatars, email, and social.
