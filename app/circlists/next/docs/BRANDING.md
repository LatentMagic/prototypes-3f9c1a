# Branding

Brand pack lives in [`../brand/`](../brand/circlists-brand.md) — palette, mark, wordmark, lockup, type.

**Source of truth is the company wiki** — [github.com/LatentMagic/wiki/tree/main/wiki/products/circlists/brand](https://github.com/LatentMagic/wiki/tree/main/wiki/products/circlists/brand). `brand/` is a manually-synced shell copy for this prototype to reference, not maintained independently here.

The copy is the full wiki directory, as-is, for transparency — including the Python generators (`brand/scripts/`) that regenerate the SVGs and raster set. This prototype only ever reads the shipped assets in `brand/assets/`; the scripts aren't run from here.
